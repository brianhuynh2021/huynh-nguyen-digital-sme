# Huynh Nguyen Digital SME

Static landing page for Huynh Nguyen's SME digital transformation services.

## Deploy

The site is deploy-ready from the repository root:

- `index.html`
- `assets/`

For GitHub Pages, publish from the `main` branch root.

## Local Preview

```bash
python3 -m http.server 4173
```

Then open `http://127.0.0.1:4173/`.
